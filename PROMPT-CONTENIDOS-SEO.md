## REGLAS DE REDACCIÓN

1. **Bilingüe FR/ES:** Títulos y CTAs en francés, descripciones y explicaciones en español
2. **Keyword density:** 1-2% para keyword principal, mencionar en H1, primer párrafo, conclusión
3. **Internal linking:** Cada página enlaza mínimo a 2-3 páginas internas
4. **Headings:** H1 único por página, H2 para secciones, H3 para subsecciones
5. **Meta descriptions:** Siempre incluir keyword + beneficio + implícito CTA. Máximo 155 chars
6. **Images:** Alt text descriptivo con keyword cuando sea natural
7. **Schema.org:** Organization en home, Course en cursos, Article en blog/recursos, FAQPage en FAQ
8. **URLs:** Limpias, en español, guiones bajos no → guiones sí
9. **Mobile first:** Párrafos cortos (3-4 líneas), listas, tablas, scannable
10. **No keyword stuffing:** Escribir para personas primero, Google segundo
11. **E-E-A-T:** Mostrar experiencia, expertise, autoridad y confianza (credenciales de profesoras, años de experiencia, examinadoras oficiales)
12. **Fresh content:** Blog y Le Mot du Jour con actualización constante (señal de frescura para Google)

## NOTAS TÉCNICAS PARA EL DESARROLLADOR

### Metadata en Next.js
Cada página debe exportar metadata con este formato:
```typescript
export const metadata: Metadata = {
  title: "Keyword principal — HolaBonjour",
  description: "Descripción con keyword en las primeras palabras, beneficio claro, max 155 chars.",
  alternates: {
    canonical: "/ruta-de-la-pagina",
  },
  openGraph: {
    title: "Keyword principal — HolaBonjour",
    description: "Misma description o variante",
    url: "https://holabonjour.es/ruta",
    siteName: "HolaBonjour",
    locale: "es_ES",
    type: "website", // o "article" para blog
  },
};
```

### Schema.org en Next.js
Añadir como `<script type="application/ld+json">` en cada página:
```typescript
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{ __html: JSON.stringify(schemaData) }}
/>
```

### Sitemap
Generar `sitemap.xml` automático con Next.js:
```typescript
// src/app/sitemap.ts
export default function sitemap() {
  return [
    { url: 'https://holabonjour.es', lastModified: new Date() },
    { url: 'https://holabonjour.es/cursos', lastModified: new Date() },
    // ... todas las páginas públicas
  ];
}
```
